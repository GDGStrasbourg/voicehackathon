Équipes

L'équipe idéale est composée de 3 à 5 personnes. Cependant, tout le monde peut venir et travailler seul ou rejoindre une équipe

L'événement est ouvert à toutes personne - développeurs (même débutant), UX Designers, passionnés, femmes et hommes - qui veut apprendre et essayer quelque chose de nouveau.

    Chaque projet doit inclure :
  • Une bonne idée : excitante, créative, innovante ou unique
  • Un design conversationnel : une conversation humaine intuitive

Nous attendons un prototype fonctionnel. Chaque équipe devra présenter son projet devant notre jury. Les critères de notation sont : innovation, ergonomie et créativité.

#VoiceHackathon

Voice Hackathon est le premier hackathon international dédié à la voix. C'est un hackathon organisé par le GDG Strasbourg une association indépendante de droit local.

Prix

Pour l'équipe remportant le 1er Prix

Le GDG Strasbourg offre à chaque membre de l’équipe 1 billet d’une valeur de 40€ valable pour le Devfest Strasbourg se déroulant le 6 Novembre 2019. Pour récupérer ce gain, le GDG Strasbourg fournit un code promotionnel qui permet l’achat du billet pour 0 (zéro) euro. Le billet doit être acheté sur la plateforme de vente avant le 6/11/2019 et dans la limite des places disponibles.

Kiwi.com offre à l’équipe gagnante du premier prix la possibilité d’envoyer une vidéo de présentation du projet réalisé durant le hackathon. D’une durée de deux minutes, cette vidéo doit faire la démonstration de l’application vocale.
Un jury composé de membres de Kiwi.com et de Google sélectionne sur la base de ces vidéos dix (10) équipes qui remporterons une bon de réduction d’une valeur de 500€ à utiliser sur le site kiwi.com

Pour l'équipe remportant le 2em Prix

Le GDG Strasbourg offre à chaque membre de l’équipe 1 (un) billet d’une valeur de 40€ valable pour le Devfest Strasbourg se déroulant le 6 Novembre 2019. Pour récupérer ce gain, le GDG Strasbourg fournit un code promotionnel qui permet l’achat du billet pour 0 (zéro) euro. Le billet doit être acheté sur la plateforme de vente avant le 6/11/2019 et dans la limite des places disponibles.

Pour l'équipe remportant le 3em Prix

Le GDG Strasbourg offre à chaque membre de l’équipe 1 code de réduction permettant de bénéficier d’une réduction de 50% sur le prix d’un billet au tarif Standard (40€). Pour récupérer ce gain, le GDG Strasbourg fournit un code promotionnel qui permet l’achat du billet pour 20 euros. Le billet doit être acheté sur la plateforme de vente avant le 6/11/2019 et dans la limite des places disponibles.

Pour les autres participants

Le GDG Strasbourg offre, à toute personne n’ayant remporté aucun prix et ayant présenté un projet durant le pitch final devant le jury, un code de réduction permettant de bénéficier d’une réduction de 25% sur le prix d’un billet au tarif Standard (40€). Pour récupérer ce gain, le GDG Strasbourg fournit un code promotionnel qui permet l’achat du billet pour 30 euros. Le billet doit être acheté sur la plateforme de vente avant le 6/11/2019 et dans la limite des places disponibles.
